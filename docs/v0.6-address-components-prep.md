# v0.6 Address Components Preparation

Status: preparation only. This note identifies current surfaces and likely files
for v0.6 address component work. It does not implement address components.

## Current Address Display Surfaces

- `src/utils/format.ts` owns the current `formatAddress` helper.
- `src/utils/display-labels.ts` formats mandate holders and graph holder nodes.
- `src/wallet/WalletStatus.tsx` renders the connected wallet address.
- `src/features/organizations/OrganizationsPage.tsx` renders organization admin addresses in the table.
- `src/features/organizations/OrganizationOverviewPage.tsx` renders admin addresses and created transaction hashes.
- `src/features/governance/GovernancePage.tsx` renders mandate holder rows through `mandateDisplay`.
- `src/features/proposals/ProposalDetailsPage.tsx` renders proposal creator and target addresses.
- `src/features/proposals/ProposalsPage.tsx` renders proposal creator addresses.
- `src/features/proposals/RouteExplanationPanel.tsx` renders approval and veto actor addresses.
- `src/features/diagnostics/DiagnosticsPage.tsx` renders configured contract addresses, cursor addresses, and projection error addresses.
- `src/features/setup/SetupDraftPreview.tsx` and `src/features/setup/SetupExecutionPanel.tsx` render draft and indexed setup authority addresses.

## Current Address Inputs

- `src/features/setup/SimpleDaoPlusDraftForm.tsx` renders raw text inputs for organization admin and executor holder addresses.
- `src/features/proposals/CreateProposalPage.tsx` renders a raw target address input for custom proposal targets.

## Current Holder List Textareas

- `src/features/setup/SimpleDaoPlusDraftForm.tsx` contains `AddressListField`, a raw `textarea` used for:
  - General Council holder addresses;
  - Treasury Committee holder addresses;
  - Security Council holder addresses.

## Likely First Files To Touch

- Add shared address primitives under `src/ui/address/` or a similarly local UI component boundary.
- Update `src/utils/format.ts` only for shared formatting helpers that remain useful outside React components.
- Replace display-only address calls in organization, governance, proposal, diagnostics, setup preview, and setup execution screens with `AddressDisplay`.
- Replace single address inputs in setup and proposal creation flows with `AddressInput`.
- Replace `AddressListField` in `SimpleDaoPlusDraftForm.tsx` with a reusable multi-address input after the shared primitive exists.
- Extend `src/styles/global.css` only after the theme token shape is coordinated with `@isonia/theme-default`.
